# Japan Airfare Tracker (UI MVP)

Front-end UI prototype for the Japan Airfare Tracker MVP. It reflects the PRD scope: LAX to Tokyo searches for December 2026, fare watches, price history, and alerts.

## Run locally

- Install dependencies: `npm install`
- Start dev server: `npm run dev`
- Build for production: `npm run build`
- Preview build: `npm run preview`
